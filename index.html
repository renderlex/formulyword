📷

Розпізнавання з фото

Відкрийте зображення або вставте скріншот з буфера обміну. Виділіть мишкою потрібну формулу.

🧠

Нейромережа pix2tex

AI-модель розпізнає формулу та повертає точний LaTeX-код. Підтримує складні математичні вирази.

⌨️

Ручне введення LaTeX

Введіть або відредагуйте формулу вручну. Миттєвий перегляд результату перед збереженням.

📋

Копіювання у Word

Скопіюйте формулу в буфер обміну та вставте у Word через Ctrl+V як нативну формулу.

💾

Збереження у .docx

Збережіть одну або кілька формул у Word-файл. Підтримується пакетний режим.

🌐

Двомовний інтерфейс

Українська та англійська мови. Перемикання одним кліком.

pipeline.py 
# Formula conversion pipeline
from pix2tex import LatexOCR
import latex2mathml.converter

image  = load_image("formula.png")
latex  = LatexOCR()(image)          # "\\frac{a}{b}"
mathml = latex2mathml.converter.convert(latex)
omml   = xslt_transform(mathml)    # MML2OMML.XSL
doc    = insert_to_word(omml)       # native equation!

Розпізнати з фото
Ввести LaTeX
  1. $Натисніть «Відкрити фото» або «Вставити з буфера» (для скріншотів)
  2. $Виділіть мишкою одну формулу на зображенні
  3. $Натисніть «Розпізнати» — програма поверне LaTeX-код
  4. $Перевірте та відредагуйте LaTeX у текстовому полі
  5. $«Перевірити» — побачите прев'ю формули
  6. $«Зберегти у Word» або «Копіювати формулу» (Ctrl+V у Word)
  1. $Введіть LaTeX-формулу вручну в текстове поле
  2. $Натисніть «Перевірити» для перегляду
  3. $«Зберегти у Word» або «Копіювати формулу»
  4. $Пакетний режим: додайте кілька формул кнопкою «До списку», потім збережіть усі разом
